# Proyecto de prueba técnica Playlist

Proyecto de gestión de playlists desarrollada como prueba técnica que incluye un Backend en Java con Maven, JPA, Spring Security, expuesto atravez de un API REST. Un Fronent en ANgular y utilizando como base de datos H2, Las pruebas del API son realizadas en Postman

# Tecnologias

### Backend

- Java 21
- Maven
- Spring Boot
- JPA
- API REST

### Frontend

- Angular 21
- CSS
- HTML
